aspcore.filterdynamic.FilterSumDynamic_python

class aspcore.filterdynamic.FilterSumDynamic_python(ir)

Bases: object

Implements a time-varying convolution y(n) = sum_{i=0}^{I-1} h(i, n) x(n-i)

Use the method update_ir to change h(i,n) between using the process method to filter the signal. If update_ir is not called, the last ir is used. If update_ir is called twice, the first ir is forgotten.

Parameters:

ir (ndarray of shape (num_in, num_out, ir_len))
